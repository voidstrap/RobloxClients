# RobloxClients
For Voidstraps built in-Roblox Revival system, plan to just make Voidstrap more fun.
Roblox clients (2007 to 2013) and the local client server used by Voidstrap.

## Layout

- `engine.zip` : the local client server binaries plus shared assets (what voidstrap itself uses)
- `engine/` : the same client server binaries loose, for reference
- `maps.zip` : base maps, for all clients of roblox
- `clients/<version>.zip` : one archive per client version
- `manifest.json` : index of everything with sizes

 Not affiliated with Roblox Corp, just a simple Roblox Revival for Voidstrap.
